# AcWing-788

#AcWing 788. Number of reverse pairs <br>
Given an integer sequence of length n, please calculate the number of inverse pairs in the sequence. <br>
The definition of reverse order pair is as follows: for the ith and jth elements of a sequence, if i<j and a [i]>a [j] are satisfied, it is a reverse order pair; Otherwise, it is not. <br>
<br>
#Input format <br>
The first row contains the integer n, which represents the length of the sequence. <br>
The second row contains n integers, representing the entire sequence. <br>
<br>
#Output format <br>
Outputs an integer representing the number of inverse pairs. <br>
<br>
#Data range <br>
1 ≤ n ≤ 100000, the value range of elements in the sequence [1,109]. <br>
<br>
#Input sample: <br>
6 <br>
2 3 4 5 6 1 <br>
<br>
#Sample output: <br>
5 <br>
<br>
<br>

#AcWing 788.逆序对的数量 <br>
给定一个长度为 n 的整数数列，请你计算数列中的逆序对的数量。 <br>
逆序对的定义如下：对于数列的第 i 个和第 j 个元素，如果满足 i < j 且 a[i] > a[j] ，则其为一个逆序对；否则不是。 <br>
<br>
#输入格式 <br>
第一行包含整数 n ，表示数列的长度。 <br>
第二行包含 n 个整数，表示整个数列。 <br>
<br>
#输出格式 <br>
输出一个整数，表示逆序对的个数。 <br>
<br>
#数据范围 <br>
1 ≤ n ≤ 100000 ，数列中的元素的取值范围 [1,109] 。 <br>
<br>
#输入样例：<br>
6 <br>
2 3 4 5 6 1 <br>
<br>
#输出样例： <br>
5 <br>
